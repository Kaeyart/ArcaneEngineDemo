# Validation Checklist — 3.1.0-alpha.1.2

1. Open Unity 6000.5.2f1 and confirm compilation completes.
2. Open Start Game → Create Character.
3. Click the name field and type names containing A, D, S, W, and spaces.
4. Confirm the selected class does not move while typing.
5. Select each class using the mouse.
6. Click Begin, then click Create in the confirmation dialog.
7. Confirm the character is created and loading begins.
8. Open rename/delete dialogs and confirm clicks do not affect controls behind them.
